# HR-Analytics
This project focuses on analyzing employee data to uncover key insights related to workforce trends, attrition, and performance. Using data analytics techniques, the project helps HR teams make data-driven decisions to improve employee retention and organizational efficiency.
🔍 Key Highlights:

Analyzed employee data to identify factors influencing attrition and retention
Built interactive dashboards in Power BI for real-time HR insights
Performed data cleaning, transformation, and modeling using SQL and Excel
Created KPIs such as attrition rate, employee performance, department-wise trends, and hiring analysis
Delivered actionable insights to support strategic HR decisions

📊 Tools & Technologies:

Power BI (Data Visualization)
SQL (Data Extraction & Query Optimization)
Excel (Data Cleaning & Preprocessing)

📁 Project Features:

Interactive dashboards with filters and drill-down capabilities
Department-wise and role-based analysis
Attrition trend analysis over time
Employee demographics and performance insights

🎯 Business Impact:

Helped identify key reasons for employee turnover
Improved understanding of workforce distribution and performance
Enabled data-driven HR strategies to enhance employee engagement
